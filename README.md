# ShopHub - Your Ultimate Shopping Destination

Welcome to **ShopHub**, a modern and responsive e-commerce web application designed to provide users with an amazing shopping experience. Browse through a wide variety of products, manage your wishlist, and add items to your cart with ease.

---

## 📋 About ShopHub

ShopHub is a full-featured e-commerce platform that showcases products across multiple categories including electronics, fashion, accessories, home appliances, and fitness equipment. The platform offers a seamless shopping experience with an intuitive user interface and interactive features.

### What It Does

- **Product Browsing**: Browse through a curated collection of products across multiple categories
- **Smart Search**: Search for products in real-time with instant filtering
- **Shopping Cart**: Add products to your cart and manage quantities
- **Wishlist**: Save your favorite products for later
- **Hot Deals**: View special deals and promotional items
- **Product Filtering**: Filter products by category (Electronics, Fashion, Accessories, Home, Fitness)
- **User Authentication**: Login/Register functionality for personalized shopping
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Product Ratings**: View customer ratings and reviews
- **Social Integration**: Connect with ShopHub through social media

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server or build tool required - runs directly in the browser

### Installation & Usage

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Muskan-zehra12/ShopHub.git
   cd ShopHub
   ```

2. **Open the Application**
   - Double-click `index.html` or
   - Open with your preferred web server for better performance

3. **Start Shopping**
   - Browse through the product catalog
   - Use the search bar to find specific items
   - Add products to your cart or wishlist
   - Filter by category to narrow down options
